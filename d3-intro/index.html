D3 significa Data-Driven Documents

Tú eres el encargado de programar el gráfico
con la ayuda de una sintaxis específica

SVG
Es el ❤ de D3

Usamos selectores similares a jQuery para manipular los datos
Y luego los mostramos como vectores

whatever class="chart"
d3.select(".chart")

Con una lógica tan sencilla se pueden hacer gráficos espectaculares

  • Norway the Slow Way
  • Kentucky Population Density
  • Where We Came From and Where We Went, State by State
